Welcome to ***REMOVED***

***REMOVED*** is a web application that provides insights about companies based on public data. ***REMOVED*** uses the latest deep learning models to understand a company's core operations and provides several channels to explore the competitive landscape.

Supported Features

Search

Find similar companies: give ***REMOVED*** a company name and it will identify the five next most similar companies.

Explore

Identify clusters and segments: ***REMOVED*** provides a graphical representations of company "relatedness", which can be used to explore and identify key customer segments.

Describe

Describe a company type: enter a brief description of the type of company you are looking for and ***REMOVED*** will find the closest matches.

Model

Improve model performance: ***REMOVED*** embeds abstract representations of companies in dense vectors, which can be used to enhance the feature engineering process.
