What is ***REMOVED***

***REMOVED*** is a web application designed to compare companies using insights derived from the latest deep learning models. ***REMOVED*** processes public data to learn about a companies business.

Supported Features

Search

Find similar companies. Give ***REMOVED*** a company name and it will identify the five next most similar companies based off of company-provided descriptions.

Explore

Identify clusters and segments. ***REMOVED*** provides a graphical representations of company "closeness", which can be used to explore and identify key customer segments.

Describe

Describe a company type. Enter a brief description of the type of company you are looking for and ***REMOVED*** will find the best matches.

Model

Use ***REMOVED***'s insights for feature engineering. ***REMOVED*** embeds abstract representations of company descriptions in dense vectors, which can be leveraged in the modeling process.
